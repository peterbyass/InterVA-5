This folder contains the code for InterVA5 apps, SendingData and DecodeData

SendingData
This app receives data from ODK Collect Form, Applies InterVA5, Encodes and Sends the data in an SMS, Returns the data back to ODK Collect Form as well

DecodeData
This app can Decode received encoded SMS from SendingData app. It can display the InterVA5 data in readable format within the app.
